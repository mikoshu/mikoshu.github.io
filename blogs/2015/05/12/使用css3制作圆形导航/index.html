使用css3制作圆形导航

5月 12 2015

圆形导航制作
如图所示，使用css3和html来完成一个仿手机样式的圆形导航，在网络上已经有很多相应的教程了，这里分享一些自己的体会。
首先，我们看看需要用到的属性。border-radius 这个只要有圆形，或者圆角就都需要用到；rotate(xxdeg) 2d旋转，主要用于中间扇形部分的旋转；然后是skew(xxdeg)这个属性用于将一个矩形变形成为平行四边形;
如下是步骤示意图：

首先定义我们的结构，这里我用ul和li，li里面再包裹div或者a标签，可以看出，图中大容器是一个ul，左上角的是一个li，里面包含一个div。比如我们要完成一个圆形，内部有六个字内容的导航，可以将六个li重叠用绝对定位将他们右下角定位到整个ul的中心点,使用transform-origin属性将li的旋转中心也定位到右下角。
代码如下：

ul{
    width:598px; 
    height:598px; 
    border-radius:50%; 
    background:#f5f6e6; 
    position:relative; 
    list-style:none; 
    overflow:hidden;  
    }
li{
    width:628px; 
    height:628px;  
    position:absolute;
    top:-329px; 
    left:-329px; 
    -webkit-transform-origin:100% 100%; 
    overflow:hidden;
    border-bottom:1px solid #fff;
    }

然后让每个li分别变形，并且旋转。这里要一个小小的计算，比如我将一个圆形分成了六份，那么360/6可以知道，这个大圆里面的每个扇形的角度是60°，那我们就得用skew()属性先来改变li的角度，也就是先让里变形，由于我们需要的内角是60°，那么我们就让他在x轴偏移30°就ok了。

代码如下：

li{
    width:628px; 
    height:628px;  
    position:absolute;top:-329px; 
    left:-329px; 
    -webkit-transform-origin:100% 100%; 
    overflow:hidden;
    border-bottom:1px solid #fff; 
    transform:skew(30deg);
}

这样以后，我们就让六个重叠的li和li里的div都变形成了右下角为60°的平行四边形了，接下来我们依次旋转每个li的角度，由于六个li要填满一个圆形，所以每个角度都是60的倍数从0开始。
代码如下：
html：

<ul class="nav">
 <li class="nav1">
     <div><ins></ins></div>
    </li>
    <li class="nav2">
     <div><ins></ins></div>
    </li>
    <li class="nav3">
     <div><ins></ins></div>
    </li>
    <li class="nav4">
     <div><ins></ins></div>
    </li>
    <li class="nav5">
     <div><ins></ins></div>
    </li>
    <li class="nav6">
     <div><ins></ins></div>
    </li>
</ul>

css：

.nav{
    width:598px; height:598px; 
    border-radius:50%; 
    background:#f5f6e6; 
    position:relative; 
    list-style:none;  
    margin:1000px auto;  
    }
.nav li{
    width:628px; 
    height:628px;  
    position:absolute;
    top:-329px; 
    left:-329px; 
    -webkit-transform-origin:100% 100%; 
    overflow:hidden;
    border-bottom:1px solid #fff; 
    background:#e2c58b;
    }
.nav1{  -webkit-transform: rotate(0deg) skew(30deg);}
.nav2{ -webkit-transform: rotate(60deg) skew(30deg);}
.nav3{  -webkit-transform: rotate(120deg) skew(30deg);}
.nav4{  -webkit-transform: rotate(180deg) skew(30deg);}
.nav5{  -webkit-transform: rotate(240deg) skew(30deg);}
.nav6{ -webkit-transform: rotate(300deg) skew(30deg);}

效果如下：

由于整个li变形后，li里的div等元素都会跟着相应的变形，如果要div的内容正常显示，不发生变形，那么就得将div使用skew（-30deg）属性变形回来，由于li变形是skew（30deg）所以里面的元素要变形回原来转态则为skew（-30deg）。同时div还得旋转角度，这里同上，旋转的角度为-60deg
代码如下：

.nav li div{
    width:578px; 
    height:578px; 
    position:absolute;
    right:-289px; 
    bottom:-289px; 
    border-radius:50% 50% 0 0; 
    -webkit-transform:skew(-30deg) rotate(-60deg) ; 
    text-align:center; 
    background:#e2c58b; 
    cursor:pointer; 
    }

（这里需要注意，transform:skew(-30deg) rotate(-60deg) ;必须先进行skew变形后再旋转，不可先旋转再变形，否则整个div将无法恢复。）

接下来，我们给ul设置overflow：hidden属性就行了，可以稍微加一点过渡效果如下：
